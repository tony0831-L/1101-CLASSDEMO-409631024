# 3. 心得甘苦談

### 這次的所有項目都是手刻的,全體都有rwd設計,比較有挑戰性的是要完全不依賴第三方ui框架或是Vuejs等框架,雖然難倒是沒有甚麼難度,但要在短時間內完全一個人發想文案,畫出設計稿,手動切版等等,這還真的是不太容易。



### 使用工具類:ClipStudioPaint(插畫,圖片編輯), AdobeXD(線框稿設計與排版), Zeplin(線框稿導出為設計規範圖), VsCode(editor)。
